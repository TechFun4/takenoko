# Extension Icons

Place your extension icons here with the following names:

- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels
- `icon128.png` - 128x128 pixels

## Creating Icons

You can create icons using:
- **Design tools**: Figma, Sketch, Photoshop
- **Online generators**: https://www.logomaker.com/, Canva
- **AI generators**: DALL-E, Midjourney

## Suggested Design

- Use a piggy bank or coin icon
- Include round/circular elements (representing round-up)
- Color scheme: Purple gradient (#667eea to #764ba2)
- Keep it simple and recognizable at small sizes
